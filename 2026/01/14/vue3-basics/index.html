全面学习 Vue 3 基础
2026-01-14|技术

1. 为什么选择 Vue 3

  • 更快：编译器优化 + Proxy 响应式
  • 更轻：按需 API，Tree-shaking 友好
  • 更易维护：组合式 API 更利于逻辑复用

2. 项目初始化

目录结构要点：

  • src/main.ts：应用入口
  • src/App.vue：根组件
  • src/components/：通用组件

3. 响应式系统

  • ref 用于基础类型，.value 访问
  • reactive 用于对象/数组，自动深层代理

4. 组件与模板语法

  • 事件：@click
  • 绑定：:class、:style、v-model
  • 条件/列表：v-if / v-for

5. 组合式 API

  • computed：派生值，具备缓存
  • watch：侦听变化，适合副作用

6. 生命周期

  • onMounted：DOM 就绪
  • onUpdated：响应式更新后
  • onUnmounted：清理副作用

  • 父子：props / emit
  • 祖孙：provide / inject
  • 跨层：状态管理（Pinia）

12. 指令与插件

  • 自定义指令：focus、lazy-load
  • 插件：全局方法、全局组件注册

13. 性能优化

  • 按需路由与组件懒加载
  • KeepAlive 缓存组件
  • 合理使用 computed 避免重复计算

14. SSR 简介

  • 选择 Nuxt 3 / Vite+自建 SSR
  • SEO、首屏性能更好，复杂度提高

15. 常见陷阱与建议

  • reactive + 解构会丢失响应式，需用 toRefs
  • watch 默认惰性，深度侦听需 { deep: true }
  • 组合式 API 下，逻辑拆分为 composables，提高复用性
