数系演变：从自然数到八元数（含定义、运算与性质）
2026-01-15|edumath

引言：数的扩张之路

数学的发展史，本质上是“数系”不断扩张以解决更复杂问题的历史。每一步扩张，都伴随着逻辑的完善和物理世界的对应。

第一阶段：代数结构的起点（实数系详尽分类）

数的扩张始于对运算封闭性的追求。在进入复数域之前，实数系内部有着严密的层级。

1. 自然数 (N\mathbb{N}N)

  • 定义： 最小的数集，基于皮亚诺公理。
  • 组成： {0,1,2,3,… }\{0, 1, 2, 3, \dots\}{0,1,2,3,…}（注：现代数学通常将 0 包含在内）。

2. 整数 (Z\mathbb{Z}Z)

整数集是自然数的扩张，解决了减法的封闭性。

  • 正整数 (Z+\mathbb{Z}^+Z+)： 即 {1,2,3,… }\{1, 2, 3, \dots\}{1,2,3,…}。
  • 零 (0)： 既不是正数也不是负数，是加法单位元。
  • 负整数 (Z−\mathbb{Z}^-Z−)： 即 {−1,−2,−3,… }\{-1, -2, -3, \dots\}{−1,−2,−3,…}。
  • 性质：
    • 奇偶性： 能被 2 整除的为偶数 (2k2k2k)，不能被整除的为奇数 (2k+12k+12k+1)。
    • 质合性： 在大于 1 的正整数中，只有 1 和自身两个因数的为质数，否则为合数。

3. 有理数 (Q\mathbb{Q}Q)

有理数是可以表示为两个整数之比 pq\frac{p}{q}qp​ (q≠0q \neq 0q=0) 的数。

  • 分数 (Fractions)：
    • 真分数： 分子小于分母（如 2/32/32/3）。
    • 假分数： 分子大于或等于分母（如 5/45/45/4）。
    • 带分数： 整数与真分数的组合（如 1141 \frac{1}{4}141​）。
  • 小数 (Decimals) 与有理数的关系：
    • 有限小数： 如 0.5=1/20.5 = 1/20.5=1/2，必为有理数。
    • 无限循环小数： 如 0.333⋯=1/30.333\dots = 1/30.333⋯=1/3，必为有理数。
    • 判定准则： 凡是能写成两个整数相除的形式，或表现为有限/无限循环小数的，均为有理数。

4. 无理数 (Irrational Numbers)

无理数是不能表示为分数形式的数，即无限不循环小数。

  • 代数数 (Algebraic)： 某些整系数多项式的根（如 2\sqrt{2}2​ 是 x2−2=0x^2 - 2 = 0x2−2=0 的根）。
  • 超越数 (Transcendental)： 不是任何整系数多项式的根（如 π≈3.14159…\pi \approx 3.14159\dotsπ≈3.14159…，e≈2.71828…e \approx 2.71828\dotse≈2.71828…）。
  • 发现历史： 希帕索斯因发现 2\sqrt{2}2​ 无法表示为有理数而引发了第一次数学危机。

5. 实数 (R\mathbb{R}R)

实数是有理数与无理数的并集。

  • 连续性： 实数填满了数轴上的每一个点，没有空隙。
  • 运算： 四则运算（除零外）在实数域内是完备且封闭的。

第二阶段：维度与旋转（复数与高维数系）

5. 复数 (C\mathbb{C}C)

  • 定义： z=a+biz = a + biz=a+bi，其中 i2=−1i^2 = -1i2=−1。
  • 几何意义： 复平面。乘法对应旋转与缩放。
  • 运算性质：
    • 交换律： ab=baab = baab=ba
    • 结合律： (ab)c=a(bc)(ab)c = a(bc)(ab)c=a(bc)
    • 模： ∣z∣=a2+b2|z| = \sqrt{a^2 + b^2}∣z∣=a2+b2​
  • 意义： 代数基本定理保证了 nnn 次多项式在 C\mathbb{C}C 内必有 nnn 个根。

6. 四元数 (H\mathbb{H}H) —— 丢掉交换律

  • 定义： 1843 年由哈密顿发现，q=a+bi+cj+dkq = a + bi + cj + dkq=a+bi+cj+dk。
  • 核心公式： i2=j2=k2=ijk=−1i^2 = j^2 = k^2 = ijk = -1i2=j2=k2=ijk=−1。
  • 运算性质：
    • 非交换性： ij=kij = kij=k 但 ji=−kji = -kji=−k。
  • 应用： 3D 旋转建模（比欧拉角更稳健，避免万向节死锁）。

7. 八元数 (O\mathbb{O}O) —— 丢掉结合律

  • 定义： o=a0+∑i=17aieio = a_0 + \sum_{i=1}^7 a_i e_io=a0​+∑i=17​ai​ei​。
  • 运算逻辑（Fano 平面）：
  • 性质：
    • 非结合性： (ab)c≠a(bc)(ab)c \neq a(bc)(ab)c=a(bc)。
    • 仅有的除法代数： 只有 R,C,H,O\mathbb{R}, \mathbb{C}, \mathbb{H}, \mathbb{O}R,C,H,O 是规范赋范除法代数（Hurwitz 定理）。
  • 应用： 弦理论、高维物理对称性（如 G2G_2G2​ 李群）。

第三阶段：统一视角（Cayley-Dickson 构造）

数系的每一次倍增（从 nnn 维到 2n2n2n 维）都可以通过 Cayley-Dickson 构造 描述：
设 z=(a,b)z = (a, b)z=(a,b)，其中 a,ba, ba,b 为 nnn 维数。

  • 加法： (a,b)+(c,d)=(a+c,b+d)(a, b) + (c, d) = (a+c, b+d)(a,b)+(c,d)=(a+c,b+d)
  • 乘法： (a,b)(c,d)=(ac−dbˉ,aˉd+cb)(a, b)(c, d) = (ac - d\bar{b}, \bar{a}d + cb)(a,b)(c,d)=(ac−dbˉ,aˉd+cb)
  • 共轭： (a,b)‾=(aˉ,−b)\overline{(a, b)} = (\bar{a}, -b)(a,b)​=(aˉ,−b)

性质的丢失清单：

  1. 从 R→C\mathbb{R} \rightarrow \mathbb{C}R→C：丢失了全序性（无法比较 iii 与 0 的大小）。
  2. 从 C→H\mathbb{C} \rightarrow \mathbb{H}C→H：丢失了交换律。
  3. 从 H→O\mathbb{H} \rightarrow \mathbb{O}H→O：丢失了结合律。
  4. 从 O→十六元数(S)\mathbb{O} \rightarrow \text{十六元数}(\mathbb{S})O→十六元数(S)：丢失了除法封闭性（出现了零因子）。

总结与闭环

  1. 定义闭环： 每个数系都是前者的完备化或维度扩张。
  2. 运算闭环： 从标量到向量再到代数结构的统一。
  3. 逻辑校验： 随着维度的增加，数学性质（如交换、结合）被作为“代价”交换了更强的表达能力。
