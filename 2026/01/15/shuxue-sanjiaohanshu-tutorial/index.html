一、公式的由来：几何与投影证明

三角函数的核心在于“比例”。通过直角三角形与单位圆的结合，我们可以清晰地看到公式的诞生。

1. 为什么 sin⁡θ=对边/斜边\sin\theta = \text{对边} / \text{斜边}sinθ=对边/斜边？

证明思路： 利用相似三角形的性质。

  • 步骤：
    1. 在直角坐标系中，画一个以原点为顶点，始边在 xxx 轴正半轴的角 θ\thetaθ。
    2. 在角 θ\thetaθ 的终边上任取两点 P1(x1,y1)P_1(x_1, y_1)P1​(x1​,y1​) 和 P2(x2,y2)P_2(x_2, y_2)P2​(x2​,y2​)。
    3. 过这两点分别向 xxx 轴作垂线，得到两个直角三角形。
    4. 几何原理： 由于这两个三角形相似（AA 相似），对应边成比例：

      y1r1=y2r2\frac{y_1}{r_1} = \frac{y_2}{r_2} r1​y1​​=r2​y2​​

      （其中 rrr 为斜边长）。
  • 结论： 只要角度 θ\thetaθ 固定，对边与斜边的比值就是唯一的常数。我们将这个比值定义为 sin⁡θ\sin \thetasinθ。

2. 投影法证明 (r⋅cos⁡θr\cdot\cos\thetar⋅cosθ 与 r⋅sin⁡θr\cdot\sin\thetar⋅sinθ)

图形演示：

  • 证明步骤：
    1. 设斜边长度为 rrr。
    2. 根据三角定义，点 P(x,y)P(x, y)P(x,y) 在圆上，则 x=r⋅cos⁡θx = r \cdot \cos \thetax=r⋅cosθ（水平投影），y=r⋅sin⁡θy = r \cdot \sin \thetay=r⋅sinθ（垂直投影）。
    3. 逻辑闭环： sin⁡θ=y/r=(rsin⁡θ)/r=sin⁡θ\sin \theta = y/r = (r \sin \theta)/r = \sin \thetasinθ=y/r=(rsinθ)/r=sinθ。证明了投影与边长定义的自洽性。

九、综合应用篇：三角方程与恒等变换（闭环校验）

在解决复杂组合题时，关键在于“化同角、化同名、化同次”。

1. 三角方程解法：2sin⁡2x+3cos⁡x−3=02\sin^2 x + 3\cos x - 3 = 02sin2x+3cosx−3=0

证明与步骤演示：

  1. 统一函数名： 利用 sin⁡2x=1−cos⁡2x\sin^2 x = 1 - \cos^2 xsin2x=1−cos2x。
    • 方程变为：2(1−cos⁡2x)+3cos⁡x−3=02(1 - \cos^2 x) + 3\cos x - 3 = 02(1−cos2x)+3cosx−3=0。
  2. 化简： −2cos⁡2x+3cos⁡x−1=0-2\cos^2 x + 3\cos x - 1 = 0−2cos2x+3cosx−1=0 → 2cos⁡2x−3cos⁡x+1=02\cos^2 x - 3\cos x + 1 = 02cos2x−3cosx+1=0。
  3. 因式分解： (2cos⁡x−1)(cos⁡x−1)=0(2\cos x - 1)(\cos x - 1) = 0(2cosx−1)(cosx−1)=0。
  4. 求解： cos⁡x=1/2\cos x = 1/2cosx=1/2 或 cos⁡x=1\cos x = 1cosx=1。
    • 当 cos⁡x=1/2\cos x = 1/2cosx=1/2 时，x=2kπ±π/3x = 2k\pi \pm \pi/3x=2kπ±π/3。
    • 当 cos⁡x=1\cos x = 1cosx=1 时，x=2kπx = 2k\pix=2kπ。

闭环检查：

  • 代入原方程验证：
    • x=0x=0x=0 时，2(0)2+3(1)−3=02(0)^2 + 3(1) - 3 = 02(0)2+3(1)−3=0（成立）。
    • x=π/3x=\pi/3x=π/3 时，2(3/2)2+3(1/2)−3=3/2+3/2−3=02(\sqrt{3}/2)^2 + 3(1/2) - 3 = 3/2 + 3/2 - 3 = 02(3​/2)2+3(1/2)−3=3/2+3/2−3=0（成立）。
  • 逻辑校验： 统一了函数名为 cos⁡x\cos xcosx，并正确处理了二次项。

2. 恒等变换组合：已知 tan⁡α=2\tan \alpha = 2tanα=2，求 sin⁡α+3cos⁡αsin⁡α−cos⁡α\frac{\sin \alpha + 3\cos \alpha}{\sin \alpha - \cos \alpha}sinα−cosαsinα+3cosα​

技巧： “弦化切”，即分子分母同时除以 cos⁡α\cos \alphacosα。

  1. 变形： (sin⁡α+3cos⁡α)/cos⁡α(sin⁡α−cos⁡α)/cos⁡α=tan⁡α+3tan⁡α−1\frac{(\sin \alpha + 3\cos \alpha)/\cos \alpha}{(\sin \alpha - \cos \alpha)/\cos \alpha} = \frac{\tan \alpha + 3}{\tan \alpha - 1}(sinα−cosα)/cosα(sinα+3cosα)/cosα​=tanα−1tanα+3​。
  2. 代入： 2+32−1=5\frac{2 + 3}{2 - 1} = 52−12+3​=5。

闭环检查：

  • 除法前提： 由于 tan⁡α=2\tan \alpha = 2tanα=2，$ \alpha \neq k\pi + \pi/2$，所以 cos⁡α≠0\cos \alpha \neq 0cosα=0，除法合法。
  • 一致性： 无论是设 sin⁡α=2k,cos⁡α=k\sin \alpha = 2k, \cos \alpha = ksinα=2k,cosα=k（kkk 为缩放因子）还是弦化切，结果均为 5。

3. 复杂证明题：证明 1+sin⁡2θ+cos⁡2θ1+sin⁡2θ−cos⁡2θ=cot⁡θ\frac{1+\sin 2\theta + \cos 2\theta}{1+\sin 2\theta - \cos 2\theta} = \cot \theta1+sin2θ−cos2θ1+sin2θ+cos2θ​=cotθ

步骤演示：

  1. 倍角展开：
    • 分子：1+2sin⁡θcos⁡θ+(2cos⁡2θ−1)=2cos⁡θ(sin⁡θ+cos⁡θ)1 + 2\sin \theta \cos \theta + (2\cos^2 \theta - 1) = 2\cos \theta(\sin \theta + \cos \theta)1+2sinθcosθ+(2cos2θ−1)=2cosθ(sinθ+cosθ)。
    • 分母：1+2sin⁡θcos⁡θ−(1−2sin⁡2θ)=2sin⁡θ(cos⁡θ+sin⁡θ)1 + 2\sin \theta \cos \theta - (1 - 2\sin^2 \theta) = 2\sin \theta(\cos \theta + \sin \theta)1+2sinθcosθ−(1−2sin2θ)=2sinθ(cosθ+sinθ)。
  2. 约分： 2cos⁡θ(sin⁡θ+cos⁡θ)2sin⁡θ(sin⁡θ+cos⁡θ)=cos⁡θsin⁡θ=cot⁡θ\frac{2\cos \theta(\sin \theta + \cos \theta)}{2\sin \theta(\sin \theta + \cos \theta)} = \frac{\cos \theta}{\sin \theta} = \cot \theta2sinθ(sinθ+cosθ)2cosθ(sinθ+cosθ)​=sinθcosθ​=cotθ。

闭环检查：

  • 公式选择： 为了消去常数 111，分子选择了 2cos⁡2θ−12\cos^2 \theta - 12cos2θ−1，分母选择了 1−2sin⁡2θ1 - 2\sin^2 \theta1−2sin2θ。
